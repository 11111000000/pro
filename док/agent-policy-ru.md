# Локальная политика агента для ПРО

Эта политика действует только в репозитории ПРО.

## Язык коммитов

- Commit message должен быть сгенерирован агентом по-русски.
- Латиница допустима, если это технические токены, имена файлов, пути, идентификаторы или неизбежные термины.
- Смысл, формулировка и доминирующая часть сообщения должны быть на русском языке.
- Перед коммитом агент обязан проверить сообщение через локальный helper `scripts/commit-ru.sh`.
- Для принудительного контроля в этой репе нужно один раз выполнить `scripts/install-local-hooks.sh`.
- Локальный hook находится в `.githooks/commit-msg` и действует только после настройки `core.hooksPath` в этом репозитории.

## HDS

- Любое изменение выполняется по HDS: `Surface → Proof → Code → Verify → Update HOLO/Decisions`.
- Для каждого изменения нужен Change Gate.
- Если затронут `[FROZEN]`, обязателен `Migration Block`.

## TTY

- В TTY агент должен сохранять plain-text UI без лишних символов, иконок и декоративных Unicode-замен.
- В TTY предпочитай ASCII fallback для промптов, баннеров и model-line, если графические glyph'ы недоступны.
- В TTY не включай `prettify-symbols`, `org-modern`, Unicode ellipsis и декоративные glyph'ы в prompt/banner.

## Границы

- Эти правила не распространяются на другие репозитории.
- Вне этой репы агент не должен считать эту политику глобальной.
