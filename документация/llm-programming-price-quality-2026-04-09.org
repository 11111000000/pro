** Резюме
Для программирования по соотношению цена/качество сейчас лучше всего смотрятся:
- =deepseek/deepseek-v3.2= как лучший общий баланс для кода, reasoning и agentic-workflows.
- =qwen/qwen3-coder= как лучший «чисто кодовый» вариант.
- =openai/gpt-oss-120b= как очень сильный бюджетный reasoning/tool-use вариант.
- =google/gemini-3-flash-preview= как лучший long-context + multimodal помощник для интерактивной разработки.
- =google/gemma-4-26b-a4b-it= / =google/gemma-4-31b-it= как хорошие multimodal-варианты для UI/скриншотов.

Если нужен один дефолт:
- для платного API: =deepseek/deepseek-v3.2=
- для free/OpenRouter: =qwen/qwen3-next-80b-a3b-instruct:free=
- для чистого кода без лишней «болтовни»: =qwen/qwen3-coder=

** Как оценивалось
Смотрел на:
- цену входа/выхода на 1M токенов,
- длину контекста,
- сигналы качества с OpenRouter pages и rankings,
- пригодность для codegen, refactor, debugging, tool use,
- multimodal-возможности для ревью UI и скриншотов.

** Источники
- OpenRouter model pages.
- OpenRouter rankings.
- Текущий локальный список моделей и тарифов в репозитории.

** Выводы
1. Лучшее общее соотношение цена/качество для кода и агентных задач: =deepseek/deepseek-v3.2=.
2. Лучший кодовый specialist: =qwen/qwen3-coder=.
3. Лучший ультрабюджетный reasoning/tool-use: =openai/gpt-oss-120b=.
4. Лучший multimodal для разработки с UI/скриншотами: =google/gemma-4-26b-a4b-it=.
5. Лучший long-context interactive assistant: =google/gemini-3-flash-preview=.
6. Лучший дешёвый кодовый вариант в AITUNNEL: =qwen3-coder-next=.

** Практический shortlist
Для повседневного программирования держать 3-4 модели:
- =deepseek/deepseek-v3.2= как основной платный дефолт.
- =qwen/qwen3-coder= как code-specialist.
- =openai/gpt-oss-120b= как дешёвый reasoning fallback.
- =google/gemini-3-flash-preview= для длинных контекстов и multimodal.
- =qwen3-coder-next= как дешевый OpenAI-compatible кодовый вариант в AITUNNEL.

** Free-альтернатива из текущего проекта
Если нужен бесплатный вариант, локальный конфиг уже хорошо отражает текущий рынок:
- =qwen/qwen3-next-80b-a3b-instruct:free= как общий дефолт.
- =qwen/qwen3-coder:free= как лучший free для кода.
- =openai/gpt-oss-120b:free= и =minimax/minimax-m2.5:free= как сильные fallback-опции.
- =google/gemma-4-26b-a4b-it:free= и =google/gemma-4-31b-it:free= как multimodal free.
- =qwen/qwen3-next-80b-a3b-instruct:free= как общий free default.

** Риск/ограничения
- Цены на OpenRouter меняются по провайдерам и времени.
- Rankings отражают usage, а не прямой head-to-head benchmark.
- Некоторые модели сильны на reasoning, но не всегда лучшие именно для code completion.

** Рекомендуемая следующая итерация
- Добавить в конфиг две ветки: =code-first= и =reasoning-first=.
- Привязать default к =deepseek/deepseek-v3.2= или =qwen/qwen3-next-80b-a3b-instruct:free= в зависимости от pay/free режима.
- Для multimodal ревью держать отдельный список =gemma-4-26b/31b= и =gemini-3-flash-preview=.
