** Recommended Default
:PROPERTIES:
:CUSTOM_ID: recommended-default
:END:
Recommended default for =gptel=: =qwen/qwen3-next-80b-a3b-instruct:free=

Reason: - Best quality/speed balance among current free text models. -
OpenRouter describes it as optimized for fast, stable responses. -
Strong fit for general assistant work, code help, RAG, tool use, and
long-context tasks. - Lower active parameter footprint than the largest
free models, so it is a safer default than pure "max quality" picks such
as =qwen/qwen3-coder:free= or
=nousresearch/hermes-3-llama-3.1-405b:free=.
